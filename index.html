Pipeline Overview

  • 1 Source: GitHub triggers Jenkins.
  • 2 Build: Docker image created & scanned.
  • 3 Infra: Terraform provisions AWS resources.
  • 4 Config: Ansible applies security configs.
  • 5 Deploy: Dev → Test → Prod environments.
  • 6 Monitor: Logs, metrics, and alerts.
